# ♻️ Smart Waste Management System Dashboard (Thesis Project)

## 📌 Project Overview
This project is a Smart Waste Management System dashboard designed for monitoring waste levels across multiple zones in Dhaka City.  

It provides a visual simulation of smart bins distributed across zones and helps demonstrate how a smart city waste collection system can be managed using real-time monitoring concepts.

This project is developed as part of a CSE thesis to present a low-cost, scalable, and structured waste management solution.

---

## 🎯 Objectives
- Divide Dhaka City into multiple operational zones
- Simulate smart bins within each zone
- Monitor waste accumulation visually for each bin
- Identify bins that require urgent waste collection
- Support optimized truck dispatch decisions
- Demonstrate smart city waste management concept

---

## 🧠 Key Features
- 5-zone city segmentation (Zone 1 – Zone 5)
- 10 bins per zone (total 50 bins simulation)
- Multi-waste category visualization:
  - Paper
  - Plastic
  - Metal
  - Organic
  - Liquid waste
- Bin-level waste monitoring UI
- Smart dispatch recommendation panel
- Zone-based map visualization using Leaflet.js
- Interactive dashboard interface

---

## 💻 Technologies Used
- HTML5

---

## 🗺️ System Description
- Each zone represents a defined geographical area of Dhaka City
- Each zone contains 10 smart bins
- Each bin tracks multiple waste categories
- System highlights bins requiring urgent collection
- Map shows zone-level spatial distribution

---

## 🚛 Truck Dispatch System
The dashboard includes a simulated dispatch recommendation system that:
- Identifies overfilled bins
- Groups them by zone
- Suggests collection priority
- Helps optimize waste collection routes
